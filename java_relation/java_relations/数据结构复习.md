# 图

## 概念

- 顶点（vertex）
- 边(edge)
- 无向图
- 有向图
- 带权图
- 路径

## 图的表示方式

1. 邻接矩阵：以顶点的数量为矩阵的长和宽，有边则置为1，其他保持为0。
2. 邻接表：数组+链表：以图的顶点进行编号，并存入数组中，当有顶点当前顶点相连，则插入链表中。
